Contributing to Gridcoin

Gridcoin is an open source project which means everyone can contribute in some way.
Select a tab below to explore the various ways to contribute to gridcoin.

Core Wallet Software Gridcoin Website Community & Outreach Governance

There are many different ways you can contribute to the development of the core wallet software. The most important task is Code Development.
The Core Wallet Software can be broken down into a number of sub systems, each of which require some level of development and maintenence.

  • RPC System - The RPC system allows external applications to query the 'Core Wallet' for blockchain data, wallet data and much more.
  • Gridcoin Oracle - The Gridcoin Oracle gathers, compresses and distributes the statistics data required for each node to compute research rewards in a trustless and decetralized manner.
  • Blockchain Protocal - The Blockchain Protocal exists.
  • Graphical User Interface (GUI) - The GUI allows our users to interact with the internal functions of the core wallet with ease and simplicity.

You can also contribute by giving feedback on various issues, Reporting bugs, Developing assets for use in the wallet, Updating build guides and the wiki or by simply voicing your opinion.

Development discussion takes place in the #Development channel on Slack.

There are many different ways you can contribute to the development of the Gridcoin Website.
The main aspects of development are:

  • Layout & Design - You can contribute to improving the layout and design of the Gridcoin Website.
  • Content - You can contribute by keeping content up to date and by adding to and improving existing content.
  • Grahpical assets - You can help develope assets for use on the site that match the current overall design of the Gridcoin Website.

You can also contribute by giving feedback on various issues and reporting bugs.

Development discussion takes place in the #Web-Dev channel on Slack.
