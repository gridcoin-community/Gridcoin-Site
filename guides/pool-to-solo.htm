Removing the pool from BOINC

  1. Use the menu item in BOINC "stop using <pool's name>"
  2. Switch to advanced view if you are not already using it (View → Advanced View...)
  3. Remove all projects from the projects tab in BOINC
    You will not be able to receive rewards if you continue crunching for the pool's account

Signing up for projects

Note: If you are an existing BOINC user and use an account manager such as BAM, you can add projects via your account manager so long as accounts at the projects all use the same e-mail address (BAM's default).

  1. Create personal accounts on all desired projects using the same email
  2. Attach desired projects to your BOINC client (Tools → Add project...)
  3. Make sure that your CPIDs are consistent across your projects (it can be checked on the project's site/your account manager/the Gridcoin Researcher Wizard)
  4. If they are not consistent then you're suffering from a split CPID. Instructions to fix it can be found below.
  5. Proceed to Step 2 if your CPIDs are all the same.
    DO NOT CONTINUE to advertising your beacon if you have a split CPID.
    You would likely need to readvertise your beacon which would waste your time.

Resolving a split CPID

  1. Attach all desired projects to the same BOINC client, if they are not already (Attach all projects - even if you plan to crunch it on a different host)
  2. Force an update of all your projects from the projects tab
  3. Check your CPIDs on the projects' site
  4. If they are still not consistent wait a few minutes before attempting to update them again
