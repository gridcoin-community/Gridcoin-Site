IMPORTANT NOTICE

You will need approx 1-2 GRC to get started with solo crunching, which you can obtain from a faucet, a friendly user on our discord server, or an exchange. To collect accrued rewards, you must stake a block or use MRC (manual reward claim) to have somebody stake for you. MRC does have some fees, and the less often you make MRC requests the lower the fees will be. Once you have sufficient GRC (approx 5,000 to stake once a month on average), you will no longer need to make MRC claims as your wallet will be able to regularly stake blocks on its own.

Don't Want BOINC/Folding@Home rewards?

Switch to the Non-Cruncher guide. You do not need to install or run BOINC in order to use GRC as a cryptocurrency.

BOINC Introduction

To earn Gridcoin, you need to crunch whitelisted BOINC projects or Folding@home. You must install the BOINC client and select one or more BOINC projects. Note that if you are intending to only crunch Folding@home (a non-BOINC project), you must still attach to a BOINC project to obtain your beacon & CPID. You can set the project to "no new tasks" in BOINC if you don't want to crunch it.

For getting Gridcoin, see step 3. You can also start out by pool mining which does not require staking/MRC and does not support Folding@home.

Install BOINC

Go to the BOINC website, download and install BOINC for any computer you want to run BOINC on. If you don't have projects that use VirtualBox, BOINC+VirtualBox is not needed.
