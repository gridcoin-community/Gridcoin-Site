BOINC Introduction

To earn Gridcoin, you need to crunch whitelisted BOINC projects or Folding@home. BOINC is a tool which connects you to various research projects incentivized by Gridcoin. You must install the BOINC client and select one or more BOINC projects. Note that Folding@home (a non-BOINC project) has some different steps required to earn GRC rewards, please see this guide before proceeding.

Install BOINC

Go to the BOINC website, download and install BOINC for any computer you want to run BOINC on. If you'll be crunching with multiple devices, please see our FAQ.

Windows users: Download the BOINC + Virtualbox option to ensure you can crunch the widest variety of projects possible.
Linux users: BOINC install packages are currently broken please use this workaround.
