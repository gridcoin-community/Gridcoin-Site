Ways to Get Gridcoin

If you don't want to buy any Gridcoin, you can get some through the faucets or through pool mining.
On most centralized exchanges you need to first buy Bitcoin in order to buy Gridcoin, except from flyp.me where you can use other cryptocurrencies.

Free/Starter Gridcoin

  • Use the Gridcoin Switzerland Faucet
  • Use the GridcoinStats Faucet
  • Join the discord server and register with the wallet bot to use the %faucet command
  • Visit uscore.net, receive GRC by registering for their Gridcoin/BOINC related newsletter.
  • Join Steemit and Whaleshares, post original Gridcoin content then exchange rewards for Gridcoin.
  • Work on Gridcoin development bounties. See the lists here and here
