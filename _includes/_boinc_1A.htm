Not interested in earning BOINC rewards?

Switch to the Non-Cruncher guide. You do not need to install or run BOINC in order to use GRC as a cryptocurrency.

BOINC Introduction

In order to earn Gridcoin, you need to crunch one or more whitelisted BOINC projects. You must install the BOINC client. A video version of this guide is available.

Install BOINC

Navigate to the BOINC website, download and install the appropriate BOINC client on each of the computers you wish to crunch BOINC work upon. To be able to crunch the widest variety of BOINC projects, we suggest the BOINC + Virtualbox bundle.
